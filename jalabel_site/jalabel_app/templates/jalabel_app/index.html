Description.

Dashboard

URL : http://origin-url/dashboard/
현재 서버에 저장되어 있는 이미지들을 보여줍니다. 각 이미지들은
  • 양품 : 옅은 녹색
  • 불량 : 옅은 적록색
  • 연기 : 진한 적색
  • 미정 : 진한 청록색
    • 으로 표시됩니다. 이미지를 클릭하면 상세사항을 확인할 수 있습니다. 우측 상단의 검색 버튼을 누르면, Lot number를 입력할 수 있으며, 입력 후 Enter를 치면 검색결과가 뜹니다.

    Capture

    URL : http://origin-url/capture/
    카메라가 연결되어 있으면 영상을 캡쳐하며, 좌측 상단의 파일 선택을 통해 컴퓨터에 있는 파일을 업로드 할 수 있습니다. Capture 버튼을 누르면 이미지가 서버로 업로드되며 약 1~2초 후에 AI를 통해 검출된 불량 부분을 보여줍니다.

    불량 표시 및 저장

    Dashboard에서 이미지 클릭 후 또는 Capture에서 Capture 후에 불량 부분을 확인 및 수정할 수 있으며, 이미지를 '불량', '양품' '나중'으로 분류하여 저장할 수 있습니다.

    불량 확인 및 수정

  • Shift + 마우스 휠 : 영상 확대 축소
  • Shift + 좌 클릭 & 드래그 : 영상 이동
  • Ctrl + 좌 클릭 : 다각형으로 불량 부분 표시 시작
  • Ctrl + 우 클릭 : 다각형 완성
  • Alt + 좌 클릭 : 지우개(커서가 사각형으로 변하며 사각형 내부의 불량을 삭제)
  • Alt + 마우스 휠 : 지우개 크기 변경

    • 분류

  • Delete(A) : 업로드된 이미지를 삭제
  • Perfect(S, Space) : 표시된 불량은 무시하고, 양품으로 분류합니다. Dashboard에서 옅은 녹색으로 표시됩니다.
  • Defect(D) : 표시된 불량대로 불량품으로 분류합니다. Dashboard에서 옅은 적색으로 표시됩니다.
  • Later(F) : 나중으로 미룹니다. Dashboard에서 청록색으로 표시됩니다.
